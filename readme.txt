How to use iManager

1. Download the zip file

2. Extract the file and copy imanager folder

3.Paste inside root directory ie in  xampp/htdocs

4. Open phpmyadmin (http://localhost/phpmyadmin)

5. Create a database with name imanager

6. Import imanager.sql file (given inside in sql folder)

7.Run the script http://localhost/imanager (frontend)

8. Signup with the Details and Login to the portal.
